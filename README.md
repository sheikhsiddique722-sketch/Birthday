# Birthday Website

A modern and responsive birthday celebration website developed using HTML, CSS, and JavaScript.
The project features an attractive user interface, smooth animations, and a creative layout designed to make birthday celebrations more interactive and memorable.

## Features

* Responsive design
* Smooth animations and effects
* Interactive birthday theme
* Modern and clean UI
* User-friendly layout

## Technologies Used

* HTML5
* CSS3
* JavaScript
